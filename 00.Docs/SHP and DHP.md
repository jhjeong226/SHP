## 1. 연구 배경 및 패러다임의 전환 (Paradigm Shift)

### 1.1. CRNP 신호 변동의 대전제 (기저 원리)
대기 상층에서 유입되는 우주선(Cosmic-ray)의 절대 강도를 스케일링하고 기압($\beta$) 등에 대한 기초 보정을 완료하고 나면, 지표면 근처의 CRNP 센서에서 측정되는 에피서멀 중성자 계수율($N$)의 잔여 변동성은 **오직 '관측 풋프린트 내 존재하는 총 수소의 양'에 의해서만 결정**된다. 

### 1.2. 기존 표준 교정식의 구조적 모순
현재 글로벌 표준인 Desilets 방정식은 다음과 같다.
$$\frac{N}{N_0} = \frac{a_0}{\theta + a_2} + a_1$$
* **기존 방식의 한계:** 형상 모수($a_0, a_1, a_2$)를 보편 상수로 고정하고, 현장의 1점 교정 데이터를 이용해 오직 $N_0$만을 역산한다.
* **The Lumping Problem:** 현장마다 정적 수소 풀(SHP: 격자수, 유기 탄소 등)의 크기가 다름에도 $a_2$를 0.115로 고정함으로써, 공간적 수소 오차를 모두 $N_0$가 떠안게 된다. 결과적으로 $N_0$가 실제 우주선 강도와 무관하게 왜곡되며, 건조기/습윤기 양 극단에서 토양 수분의 체계적 편향(Systematic bias)을 유발한다.

## 2. 수소 풀의 이원화 (Dichotomy of Hydrogen Pools)
관측소 내의 수소는 시계열적 변동성을 기준으로 철저히 분리하여 타겟팅해야 한다.

### 2.1. 동적 수소 풀 (Dynamic Hydrogen Pool, DHP)
* **정의:** 시간, 계절, 기상 이벤트에 따라 실시간으로 급격히 변동하는 수소 풀 (예: 대기 절대 습도, 수관 차단수, 생장기 식물 체내 수분).
* **처리 방안:** 대기 습도 보정($\omega$) 등을 통해 원시 데이터 전처리 단계에서 실시간 보정하여 신호에서 우선적으로 걷어낸다.

### 2.2. 정적 수소 풀 (Static Hydrogen Pool, SHP)
* **정의:** 시계열적으로 큰 변동 없이 풋프린트 내에 고정된 관측소 고유의 물리적 특성 (예: 토양 광물 격자수, 영구 생체량, 토양 유기 탄소(SOC)).
* **처리 방안:** 수분 감쇠 곡선의 '영점'을 이동시키므로, 교정 단계에서 사이트 고유의 상수인 **$a_2$(SHP 당량)**로 취급하여 반드시 역산해야 한다.

## 3. 물리적 재해석 및 핵심 변수의 정의

1. **$a_0, a_1$ (입자 물리학적 보편 상수):** 에피서멀 중성자는 수소의 출처(자유수, 격자수 등)를 가리지 않고 양성자 질량에 동일하게 반응한다(수소의 등가성). 따라서 민감도 $a_0(=0.0808)$와 점근선 $a_1(=0.372)$은 전 지구적 보편 상수로 고정된다.
2. **$a_2$ (땅의 0점):** 상수가 아니라, 해당 관측소 풋프린트에 묻힌 **SHP의 체적 수분 당량**이다. (산림/습지 = 큰 $a_2$, 사막/농경지 = 작은 $a_2$).
3. **$N_0$ (기계+하늘의 0점):** 순수 우주선 배경 강도에 **해당 센서 하드웨어 고유의 검출 효율**이 결합된 스케일링 상수이다. SHP($a_2$)의 영향을 전혀 받지 않는 독립적인 기준값이다.

> **💡 SHP가 지배하는 센서 작동 대역 (Operating Range):**
> SHP가 큰 곳(큰 $a_2$)은 중성자가 이미 많이 흡수되어 전체 $N$ 값이 점근선 부근의 **'낮은 대역'**에서 둔감하게 변동하며, SHP가 작은 곳(작은 $a_2$)은 $N_0$에 가까운 **'높은 대역'**에서 민감하게 변동한다.

## 4. 자립형 2점 교정 프레임워크 (The 2-Point Framework)
거대한 풋프린트의 공간적 이질성을 파괴적 토양 샘플링으로 극복하려는 시도(공간 대푯성 문제)를 버리고, 센서 데이터 스스로 기계의 0점($N_0$)과 땅의 0점($a_2$)을 동시 역산하는 해석적 접근.

### Phase 1: 현장 데이터 획득 및 전처리 (Pre-processing)
원시 계수율($N_{raw}$)에서 하늘과 대기(DHP)의 노이즈를 먼저 제거해야 한다.
$$N = N_{raw} \times f_p \times f_i \times f_h$$
* $f_p$: 기압 보정 (Atmospheric pressure)
* $f_i$: 입사 강도 보정 (Incoming solar/cosmic intensity via NMDB)
* $f_h$: 절대 습도 보정 (Absolute humidity)

### Phase 2: 2점 교정 연립방정식 구성
수분 차이가 명확한 임의의 건조기($\theta_1, N_1$)와 습윤기($\theta_2, N_2$)에 측정된 두 세트의 보정 데이터를 대입한다.
$$N_0 = \frac{N_1}{\frac{0.0808}{\theta_1 + a_2} + 0.372} \quad \text{--- (Eq. 1)}$$
$$N_0 = \frac{N_2}{\frac{0.0808}{\theta_2 + a_2} + 0.372} \quad \text{--- (Eq. 2)}$$

### Phase 3: 분석적 해(Analytical Solution) 산출
수식 1과 2를 등치시켜 $N_0$를 소거하면 $a_2$에 대한 2차 방정식이 도출된다.
$$A \cdot a_2^2 + B \cdot a_2 + C = 0$$
* $A = 0.372$
* $B = 0.372(\theta_1 + \theta_2) + 0.0808$
* $C = 0.372(\theta_1 \cdot \theta_2) + 0.0808 \left( \frac{N_1 \theta_1 - N_2 \theta_2}{N_1 - N_2} \right)$

근의 공식을 통해 무수치해석적(Non-numerical)으로 SHP 당량($a_2^*$)을 구한다.
$$a_2^* = \frac{-B + \sqrt{B^2 - 4AC}}{2A}$$

구해진 $a_2^*$를 다시 Eq. 1 또는 2에 대입하여 현장의 기기 기준 강도($N_0$)를 역산한다.

## 5. 검증 및 실무적 고려사항 (Validation & Considerations)

### 5.1. 모델 검증 방법론
1. **다중 포인트 교차 검증:** 총 3~4회의 교정 캠페인 수행 후, 어떤 쌍(Pair)을 조합하더라도 역산된 $N_0$와 $a_2^*$가 동일한 상수로 수렴하는지 확인하여 물리적 강건성을 증명한다.
2. **공간 상관성 분석:** 드론 초분광 기반 SOC/생체량 매핑 데이터나 SoilGrids 데이터와 역산된 $a_2^*$ 간의 선형 상관관계를 분석한다.
3. **시간적 안정성:** 장기 시계열에서 고정된 $N_0$가 편향 없는 $\theta_{CRNP}$를 산출하는지 확인한다.

### 5.2. 환경별 특별 고려사항 (e.g., KOSMOS 네트워크)
* **간척지 / 농경지 동역학:** 벼농사 지역 등 작물 생장기에는 폭발적으로 증가하는 생체량을 SHP가 아닌 시간 변동적 DHP로 취급해야 한다. 위성 생물계절학 지수(NDVI, LAI 등)와 연동되는 하이브리드 변수 $a_2(t)$ 모델링이 필요하다.
* **공간 대푯성 불일치 통제:** 현장 TDR 센서 설치 시 '시간적 안정성(Temporal Stability)'이 확보된 지점을 사전 탐색하여, 점 단위 측정이 풋프린트 전체의 평균 $\theta$를 대변할 수 있도록 통제해야 2차 방정식의 $C$항 왜곡을 막을 수 있다.
* **잔류 수분과 격자수 흡수:** 105°C 건조로 측정되지 않는 광물 격자수는 수학적으로 $a_2^*$ 텀 안에 자연스럽게 흡수됨을 인지하고 해석해야 한다.

**Conclusion:** 본 프레임워크는 CRNP 풋프린트 스케일의 공간 대푯성 한계를 센서 데이터 자체의 수학적 역산으로 돌파하며, 기존 교정의 구조적 편향을 완벽히 해소한다. 2차 방정식의 명쾌한 해석적 해는 최적화 연산 부하가 없으므로 대규모 관측망(National-scale CRNP networks)의 실시간 자동 교정 모듈로 즉각 도입이 가능하다.
